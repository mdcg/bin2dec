# Bin2Dec - Golang

## Running

Just execute the following command in this root folder:
```
> go run bin2dec.go
```

If all goes well, you will see output like this:
```
Welcome to Bin2Dec!
To exit, press 'CTRL + C'.

What value do you want to convert from binary to decimal?
```
